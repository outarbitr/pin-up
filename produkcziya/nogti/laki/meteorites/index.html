Лак, светящийся изнутри

описание

Солнечные лучи при попадании на поверхность ногтя создают эффект мерцания звезд на ногтях.

СОВЕТЫ ПО НАНЕСЕНИЮ

Шаг 1: нанесите базовый слой – это может быть основа или укрепитель для ногтей.

Шаг 2: нанесите лак в 1-2 слоя.

Шаг 3: для гель-эффекта нанесите топовое покрытие для лака.

Особенности

  • Текстура: с мелкими блестками;
  • Финиш: глянцевый;
  • Особенности: искрящаяся пудра создает эффект свечения изнутри.
